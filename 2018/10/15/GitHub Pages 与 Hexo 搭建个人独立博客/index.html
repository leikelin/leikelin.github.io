GitHub Pages 与 Hexo 搭建个人独立博客

GitHub
  • GitHub
  • Hexo
  • 博客

关于一个前端小白，如何利用GitHub Pages与 Hexo 如何创建属于自己的个人博客，记录生活的点点滴滴！

开始的啰嗦

由于我是一个java后端码农，其实对于node.js 与 hexo都比较陌生。在一次逛GitHub的机会发现了一篇搭建个人博客的文章，于是自己觉得挺有意思的，随后开始了我的博客搭建之路。

由于百度上相关的文章比较多，因此我就不再详细的说明如何一步步搭建了，我只将我参考的相关文章链接贴出来，只是给大家提供一个参考，同时在此感谢下面几位文章的原创者，文章给了我很大的帮助。

参考资料

入门级：《傻瓜都可以利用github pages建博客》比较适合不懂编程的人员

困难级：《手把手教你使用Hexo + Github Pages搭建个人独立博客》 比较适合具有编程经验的人员

我个人是上面的两种方式都尝试了，个人觉得第二种方式比较好，但是比较麻烦，特别对于不懂编程知识的人来说可能需要点时间去慢慢填坑（ps：也许我比较菜，反正遇到了不少坑）。

遇到的问题

在执行”npm install”命令时报错，我百度了很久，最终看到《 npm install 报错解决》才解决问题。

然后在定制博客主题时参考了《个人博客站点建设历程》这位博主的相关博客，从里面获益不少，在此感谢！

最后的啰嗦

由于目前Disqus评论系统在国内无法访问，虽然我找了一些相关的文章阅读了一下，但是最终还是放弃了集成评论系统，因为很多东西并不是我擅长的。后期如果机会再集成进来吧！

我就啰嗦这么多了，其它的等我想起来在啰嗦吧！当然了，如果你在构建个人博客的过程中有什么不明白的可以与我联系，我们共同学习进步！
